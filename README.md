# StrataDB

Slavi Sotirov

COS 4090A - Senior Project (AUBG 2026 Fall Semester) 

##Description of project:
Lightweight Relational Datbase engine implemented in C++.

##Scope of project:
- Page-based storage with fixed-size disk pages.
- Disk-based B+ tree storage/indexing.
- Minimal SQL subset: CREATE, INSERT, SELECT by primary key, DELETE by primary key

##Current Out of Scope:
- Joins
- Concurrency Control
- Transactions
